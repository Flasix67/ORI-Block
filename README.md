# ORI-Block

Данный скрипт блокирует все домена сайтов которые содержатся в [реестре ОРИ](https://97-fz.rkn.gov.ru/)
если вы не хотите чтобы ваши данные сливались кому нибудь.

## Предупреждаю : Если у вас плохо грузит сайт после запуска `launch.bat`, вероятно там заблокирован домен который нужен для стабильной работы сайта.

Список заблокированых сервисов/сайтов :

1. `Rutube`

2. `ГОС. сайты РКН.(просто так)`

3. `Yandex`

4. `VK`

5. `MAX`

6. `Rustore`

7. `Mail.ru`

8. `Одноклассники`

9. `Avito`

10. `Wildberries`

11. `OZON`

12. `Юла`

13. `Habr`

14. `Пикабу`

15. `Дром`

16. `Авто.ру`

17. `hh.ru`

18. `IXBT`

19. `Liveinternet`

20. `Доктор.ру`

21. `Т-Банк`

и некоторые трекеры. Скоро будет добавлено больше сайтов/доменов.
