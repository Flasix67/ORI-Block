# ORI-Block

Данный скрипт блокирует все домена сайтов которые содержатся в [реестре ОРИ](https://97-fz.rkn.gov.ru/) и сооблюдают его.

### Предупреждаю : Если у вас плохо грузит сайт после запуска, вероятно был заблокирован домен который нужен для стабильной работы сайта.

## Как использовать?
1. Для запуска нужно запустить файл `launch.bat`/`launch.sh`
2. В `list-general.txt` вы можете добавить и свои домена.
3. Для остановки нужно запустить файл `stop.bat`/`stop.sh`

## Установка на Windows
1. Скачиваете `ORI-Block-Windows.zip`, распаковываете и запускаете `launch.bat` и всё. Никаких сложностей. Тоже самое и с `stop.bat`

## Установка на Linux
1. Скачиваете `ORI-Block-Linux.zip`, распаковываете в директорию в которую вам удобно. В моем случае `~/ORI-Block`
2. Открываем терминал и пишем `cd ~/ORI-Block`
3. Для использования понадобится выдать определенные права файлам. Пишем `chmod +x ./launch.sh && chmod +x ./stop.sh`
4. И чтобы наконец таки запустить `ORI-Block` пишем `sudo bash ./launch.sh`

## Установка на Android [ROOT]
1. Скачиваете `ORI-Block-Android.zip`, распаковываете в директорию которую вам удобно. В моем случае `/sdcard/ORI-Block` или `/storage/emulated/0/ORI-Block`
2. Нам понадобится скачать и установить модуль Magisk [systemless-hosts](https://github.com/Flasix67/systemless-hosts/releases/download/1.0/systemless-hosts-empty.zip) мой специальный форк с пустым hosts.
3. Запускаем Termux и пишем `termux-setup-storage` и если что то вылазиет пишем `y`
4. Для использования понадобится выдать определенные права файлам. Пишем `chmod +x /sdcard/ORI-Block/launch.sh && chmod +x /sdcard/ORI-Block/stop.sh`
5. И чтобы наконец таки запустить `ORI-Block` пишем `sudo bash /sdcard/ORI-Block/launch.sh`
